- 매출 페이지 코드에는 "전체 기간" 옵션이 없음 (연도 셀렉트만 존재)
- **필요**: 어느 화면에서 보신 건지 스크린샷 or 페이지명
- 후보: 대시보드 / 분석 / 재무 / 다른 페이지?

### 2. P1-1 채널 표준 목록 확정
- 지금 3개 소스가 다 다름:
  - GA4 기본채널 (검색오가닉/직접/외부링크/검색광고 등)
  - CRM 리드채널 (자사채널/대표전화/블로그/오가닉_네이버 등)
  - 광고 유입소스 (구글/네이버/메타/콘텐츠/기타)
- **필요**: 통일할 표준 채널 목록 확정
- 참고 (현재 코드 기본값): `유료광고_네이버 / 유료광고_구글 / 오가닉_네이버 / 오가닉_구글 / 블로그 / 직접 / 추천 / 전화_대표 / 전화_개인 / 이메일 / 해피톡`
- 팀에서 원하는 목록으로 알려주시면 매핑 함수 (`normalizeInquiryChannel`) 확장 반영

### 3. P1-2 콘텐츠 발행일/제목 이슈
- 팀 관찰: "발행일이 안 찍히고 있고, 제목도 이상하게 나옴"
- 확인: 콘텐츠 목록 페이지 (`/marketing/content`) 어느 부분인지 스크린샷 부탁
- 지금 코드 상 발행일은 form 수동 입력 필드 (sync 로 덮지 않음)

### 4. P2-4 파이프라인 단계 순서
- 현재 코드: **신규리드 → 컨텍 → 예정 → 제안 → 미팅 → 도입직전 → 도입완료 → 이탈**
- 팀 점검 결과: "미팅이 제안 뒤에 오는 순서가 실제 영업 흐름과 맞는지 팀 확인"
- **필요**: 실제 영업 흐름 기준으로
  - (A) 지금대로 유지 (`예정 → 제안 → 미팅 → 도입직전`)
  - (B) 미팅 앞으로 (`예정 → 미팅 → 제안 → 도입직전`)
  - (C) 다른 순서
