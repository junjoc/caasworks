### 팀 논의 필요한 항목

**1. 총매출 표시 정책**
- 현재: VAT 제외 저장, UI 에 VAT 포함 병기
- 다른 정책 원하는지?

**2. CVR (전환율) 정의**
- 현재 코드: `수주 / (수주 + 실패) × 100` (결과 나온 리드의 성공률)
- 다른 정의 있나? (예: 수주 / 전체 리드)

**3. 사이트→문의 전환율** (마케팅 페이지)
- `문의클릭 / GA사이트유입 × 100`
- 파이프라인 CVR 과 이름 혼동 여지 → 표기 통일 필요

**4. ROAS**
- 현재 CRM 에 미구현 (계산 코드 없음)
- first-touch vs last-touch 정책 결정 필요
- 리드→프로젝트 자동 연결 링크 필요 (지금은 수동)

**5. 연간계약 표기**
- 매출 없는 시트 프로젝트도 CRM 리스트에 표시됨
- 별도 마킹 (예: 배지, 색상) 필요?
