## 🐛 중요 버그 수정

### 매출 ₩64,000 미스터리 (최종 해결)
- **증상**: 시트 `₩207,304,750` vs CRM `₩207,368,750` — 매월 정확히 ₩64,000 차이
- **원인**: 디자인이음새주식회사가 매월 `₩-64,000` 할인 (시트 row 34, No.26)을 마이너스 값으로 기재. Sync 스크립트가 `amount > 0` 조건으로 음수 제외
- **해결**: `amount !== 0` 으로 변경 → 음수 할인 포함
- **결과**: 12개월 모두 시트와 **1원 단위까지 완전 일치**

### 기타
- Tailwind CSS purge (src/lib 경로) → 뱃지 색상 렌더링
- auth ↔ public.users ID 불일치 시 이메일 fallback (박성언 케이스)
- 사용자 관리 편집/삭제 버튼 실수 방지 (컬럼 분리)
